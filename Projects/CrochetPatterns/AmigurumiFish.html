Amigurumi Fish

Original Pattern by Chubby Cheeks

  • MC = Magic Circle
  • ch = chain
  • sc = single crochet
  • hdc = half double crochet
  • dc = double crochet
  • tr = triple crochet
  • inc = increase
  • dec = decrease
  • sl-st = slip stitch


BODY

  • R1: MC - 6sc (6)
  • R2: inc x6 (12)
  • R3: [sc, inc] x6 (18)
  • R4: [sc,sc,inc] x2, [sc,inc] x3, [sc,sc,inc] x2 (25)
  • R5: [inc, 3sc] x2, [inc, 2sc] x3, [inc,3sc] x2 (32)
  • R6: 10sc, [inc,3sc]x3, 10sc (35)
  • COLOR CHANGE
  • R7-R14: 35sc (35)
  • R15: [5sc, dec] x5 (30)
  • INSERT EYES @ R5/R6
  • R16: [4sc, dec] x5 (25)
  • R17: [3sc, dec] x5 (20)
  • R18: [2sc, dec] x5 (15)
  • STUFF THE BODY
  • R19: [sc,dec] x5 (10)
  • R20: 2sc * incomplete round
  • CHANGE COLOR FOR TAIL
  • Close tail with 4sc accross the open gap.

TAIL

  • 3ch,tr-inc into first chain, [dc,sc] into same sticth,[sc,dc] into same stitch, tr-inc, 3ch, sl-st into last tail base

DORSAL FIN (the one on top)

  • Start around row 17 and work forward
  • 2sc,2dc,2ch,sl-st, 2ch, 4tr, 2sc, sl-st

Side Fins (x2)

  • MC - 3ch, 3dc, 3ch, sl-st
  • stitch on between R9/10
  • REPEAT FOR OTHER SIDE
